# Kubernetes Manifests (Stage 4)

These files are intentionally minimal starter templates.

Purpose:

- Provide a place to practice `ConfigMap`, `Secret`, `Deployment`, `Service`, `Ingress`, and `HPA`.
- Keep everything readable and easy to evolve as you learn.

Notes:

- **Ingress** requires an ingress controller in your cluster.
- **HPA** requires metrics-server (or equivalent) to be installed.
